[UEFI] Debug OVMF with Intel UDK Debugger

Posted on 2025-02-20 In UEFI , Intel UDK Debugger
3.9k 4 mins.

This topic is used to record how to use Intel UDK Debugger on Linux for debugging OVMF and UEFI applications.

1. Overview

This guide provides step-by-step instructions on how to use the Intel UDK Debugger on Linux to debug OVMF (Open Virtual Machine Firmware) and UEFI applications. It covers:

  • Checking GDB configuration and installing Intel UDK Debugger
  • Adding CPU breakpoints to source code
  • Building OVMF with debugging support
  • Integrating debugging with QEMU and Intel UDK Debugger
  • Debugging with GDB and Intel UDK Debugger
