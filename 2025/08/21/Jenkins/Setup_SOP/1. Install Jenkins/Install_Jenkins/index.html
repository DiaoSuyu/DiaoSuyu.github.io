[Jenkins] 1. Install Jenkins

Posted on 2025-08-21 In Jenkins
9.8k 9 mins.

This topic is used to record how to set up the Jenkins server under Ubuntu.

1. Environment

  • OS: Ubuntu 22.04.4 LTS
  • Jenkins: 2.516.2
  • Jdk: 21.0.8
  • Local ip address: 192.168.77.199

2. Configuration Steps

2.1 Update the package list

2.2 Install JDK 21

2.3 Create Jenkins_Workplace folder

2.4 Download Jenkins 2.516.2 to Jenkins_Workplace foler

2.5 Install Jenkins 2.516.2 to Jenkins_Workplace foler

  • After executing the above command, the Jenkins engine is power on, the following is the log of the default configuration.

2.6 Try to login

  • Try to entry http://192.168.77.119:8000 use browser

  • Input the initialAdminPassword which you can find from /home/ubuntu/.jenkins/secrets/initialAdminPassword and click Continue button

2.7 Install plugins

  • if no errors incur, you will see the plungins page

  • click the left one(Install suggested plugins)


2.8 Create User Account

  • if no errors incur, you will see the Create First Admin User page

  • click Skip and continue as admin button

2.9 Instance Configuration

  • if no errors incur, you will see the Instance Configuration page

  • click Save and Finish button

2.10 Finish

  • if no errors incur, you will see the ready page

  • click Save and Finish button

2.11 Login

  • Entry http://192.168.77.119:8000 use browser

  • Input the initialAdminPassword which you can find from /home/ubuntu/.jenkins/secrets/initialAdminPassword and click Sign in button

3. Strongly Recommend Configuration

3.1 Auto-start Jenkins on boot

Jenkins will automatically turn off after OS reboot, if you want Jenkins to turn on automatically every time that you turn on the device, please perform the following configuration.

  • Set the Jenkins service to start automatically when ubuntu is powered on
